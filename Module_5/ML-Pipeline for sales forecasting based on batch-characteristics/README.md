### Постановка задачи

Разработать единый, воспроизводимый ML-пайплайн для прогнозирования продаж по батч-признакм. Система будет работать в режиме пакетного внедрения с недельным циклом: прогнозы обновляются каждое воскресенье вечером. Модель обучается отдельно и загружается из S3

### Описание данных

* **Исходные данные**:
  1. Таблица `sales` — исторические данные о продажах:
    - `Store` — номер даркстора (1–45).
    - `Dept` — номер отдела внутри даркстора (1–99).
    - `Date` — неделя (дата начала недели).
    - `Weekly_Sales` — недельные продажи (руб., целевая переменная).
    - `IsHoliday` — признак праздничной недели.

  2. Таблица `stores` — информация о дарксторах:
    - `Store` — номер даркстора.
    - `Type` — тип даркстора (A, B, C).
    - `Size` — размер даркстора в условных единицах.

  3. Таблица `features` — внешние факторы:
    - `Store`, `Date` — идентификаторы.
    - `Temperature` — средняя температура за неделю.
    - `Fuel_Price` — средняя цена бензина.
    - `MarkDown1-5` — факторы рекламных акций и скидок.
    - `CPI` — индекс потребительских цен.
    - `Unemployment` — уровень безработицы.

  4. Таблица `plan` — данные для инференса, содержит те же столбцы, что и `sales`, кроме `Weekly_Sales`.  
* **Результаты**:
  1. Таблица `predictions` - прогнозные данные объема продаж
    - `date` - дата (дата начала недели)
    - `store` - номер даркстора (1-45)
    - `dept` - номер отдела внутри даркстора (1-99)
    - `predicted_weekly_sales` - прогнозное значение недельных продаж (руб.)
    - `prediction_timestamp` - дата и время прогнозирования

### Описание признаков

Для прогнозирвания недельных продаж использовались признаки:
  - `store` - номер даркстора
  - `dept` - номер отдела внутри даркстора
  - `is_holiday` - признак праздничной недели
  - `type` - тип даркстора
  - `size` - размер даркстора в условных единицах
  - `factor3` - фактор рекламных акций и скидок
  - `factor5` - фактор рекламных акций и скидок
  - `month` - месяц
  - `quarter` - квартал
  - `year` - год
  - `avg_sales_before` - средние продажи комбинаций "даркстор и отдел" за предыдущие периоды
  - `sales_1week_ago` - продажи неделю назад
  - `sales_2week_ago` - продажи 2 недели назад
  - `sales_4week_ago` - продажи 4 недели назад
  - `mean_sales_2week` - среднее продаж за последние 2 недели
  - `mean_sales_4week` - - среднее продаж за последние 4 недели

### Логика обучения

  - В качестве модели для предсказания используется CatBoostRegressor без подбора гиперпараметров
  - Метрики, полученные на валидационной выборке:
    * R2: 0.98
    * MAE: 1449.3
    * RMSE: 3060.9
  - Из-за сильного дрейфа (PSI > 0.2) были удалены признаки:
    * `factor1` - фактор рекламных акций и скидок
    * `fuel_price` - средняя цена бензина.
    * `factor4` - фактор рекламных акций и скидок
    * `unemployment` - уровень безработицы.
    * `temperature` - средняя температура за неделю.
    * `cpi` - индекс потребительских цен.
    * `factor2` - фактор рекламных акций и скидок
